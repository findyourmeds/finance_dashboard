AI로 기업 해부하기: 분석 Tool의 이해와 기초 데이터 수집을 위한 ChatGPT Colab 활용

이번 시간에는 지난 클립에서 학습한 데이터의 종류를 바탕으로, 투자 분석에 필요한 데이터를 쉽고 빠르게 얻는 방법을 살펴봅니다. 특히 주가나 재무제표와 같이 정해진 정보를 효율적으로 수집하는 데 중점을 둡니다. 이 과정은 앞으로 자동 매매 봇을 만들 때도 동일하게 적용되므로 잘 기억해두는 것이 중요합니다.

이번 클립의 내용은 전체 투자 의사결정 검토 절차 중 '종목 분석' 단계에서 주가나 뉴스 등의 데이터를 가져오는 것에 해당합니다.

**기초 데이터 수집 방법 복습: 수동 vs. 자동**

우리가 이용할 수 있는 데이터 중 야후 파이낸스(Yahoo Finance)에서 제공하는 데이터를 가져오는 방법을 연습할 것입니다.

1.  **수동 데이터 수집 (불편한 방법):**
    *   이전에 주가 데이터를 수동으로 가져올 때는 인베스팅닷컴(Investing.com) 같은 웹사이트에 접속했습니다.
    *   티커(종목 코드)를 검색한 후 과거 데이터(Historical Data) 섹션으로 이동했습니다.
    *   주가 기간을 설정하고 엑셀 파일을 다운로드하여 데이터를 얻었습니다.
    *   이 방법은 한두 회사에 대해서는 괜찮지만, 분석하려는 회사가 10개 이상일 경우 각 회사마다 반복해야 하므로 매우 비효율적입니다.

2.  **자동 데이터 수집 (효율적인 방법):**
    *   이러한 불편함을 해소하기 위해 구글 코랩(Google Colab)과 파이썬 코드를 활용하여 야후 파이낸스 데이터를 한 번에 가져와 저장하는 방법을 배웠습니다.
    *   코드를 이용하면 웹사이트에 직접 들어가 다운로드하는 대신, 자동으로 데이터를 수집할 수 있습니다.

**코랩(Colab) 활용 방법 상세 설명**

이전에 공유받은 코랩 링크는 '뷰어' 권한으로 설정되어 있어 바로 실행할 수 없습니다.

*   **코랩 파일 사용하기:**
    1.  `파일` 메뉴로 이동합니다.
    2.  `드라이브에 사본 저장`을 선택합니다.
    3.  새롭게 생성된 파일의 제목을 원하는 대로 변경하여 사용합니다.

*   **코랩 코드 실행하기:**
    1.  각 코드 셀 옆에 있는 '실행' 버튼(재생 아이콘)을 누릅니다.
    2.  처음 실행할 때는 자원을 할당하는 데 시간이 걸릴 수 있습니다 (화면에 "연결 중" 메시지 표시).
    3.  코드 실행이 완료되면 셀 왼쪽에 녹색 꺾쇠 표시가 나타납니다.

*   **데이터 입력 및 확인:**
    1.  두 번째 코드 셀에서는 분석하려는 회사의 티커 코드(예: APPLE, GOOGL 등)를 입력합니다.
    2.  기간(예: 2020년부터 2024년까지)도 설정할 수 있습니다.
    3.  이 코드를 실행하면 해당 티커에 대한 주가와 재무제표를 가져올 수 있습니다.
    4.  가져온 데이터는 코랩 화면 왼쪽의 '폴더' 버튼을 눌러 확인할 수 있습니다. 다운로드하여 엑셀 등으로 열어볼 수 있습니다.

이처럼 코랩과 파이썬을 활용하면 주가, 재무제표 등 다양한 데이터를 일괄적으로 편리하게 가져올 수 있습니다.

**ChatGPT를 활용한 데이터 수집 코드 확장**

야후 파이낸스에서 제공하는 데이터는 주가나 재무제표 외에도 방대한 양이 존재합니다. 하지만 우리가 모든 데이터를 직접 코드로 작성하여 가져오기는 어렵습니다. 이때 ChatGPT를 활용하여 필요한 코드를 생성하고 확장하는 방법을 배울 것입니다.

1.  **ChatGPT 접속 및 초기 질문:**
    *   ChatGPT에 접속하여 사용하려는 추론 모델(예: GPT-4 Mini, 또는 최신 업데이트된 모델)을 선택합니다. 다양한 모델의 특징을 파악하고 목적에 맞는 모델을 사용하는 것이 좋습니다.
    *   ChatGPT에 다음과 같이 질문합니다: "구글 코랩을 이용해서 야오 파이낸스 패키지(Yfinance package)를 이용해 기업 데이터를 가져오고 싶습니다. 어떤 데이터를 이용할 수 있는지 목록을 만들어 주세요."
    *   **주의사항:** ChatGPT가 제시하는 정보가 항상 최신 내용은 아닐 수 있습니다. 웹을 실시간으로 검색하여 알려주는 것이 아닐 수 있으므로, 정확한 최신 정보를 위해서는 다음 단계를 따르는 것이 좋습니다.

2.  **Yfinance 패키지 공식 문서 활용:**
    *   ChatGPT의 답변이 최신이 아닐 가능성을 대비하여, 구글에서 "Yfinance library" 또는 "Yfinance API reference"를 검색합니다.
    *   일반적으로 GitHub 페이지에 Yfinance 패키지의 공식 문서(API Reference)가 제공됩니다.
    *   **GitHub 문서 활용:** 개발자들은 다른 사람이 만든 패키지를 사용할 때 이 문서를 통해 사용 방법, 기능, 버그 등을 파악합니다.
    *   문서의 `다큐멘테이션(Documentation)`이나 `API 레퍼런스(API Reference)` 섹션에서 Yfinance 패키지가 제공하는 다양한 데이터 목록을 확인할 수 있습니다.
        *   예시: `파이낸셜스(Financials)`를 클릭하면 특정 회사의 재무 데이터 종류와 이를 가져오는 예시 코드를 볼 수 있습니다.
        *   **중요한 데이터: 섹터 및 인더스트리(Sector and Industry):**
            *   이는 투자 분석에서 매우 중요합니다. 이전 클립에서 티커를 이용한 멀티플 분석 시 같은 업종의 회사들과 비교해야 의미가 있다고 배웠습니다.
            *   자동 매매나 대시보드 제작 시 같은 업종의 기업 목록을 확인하는 것은 필수적인 절차입니다.
            *   미국 상장 회사는 GICS(Global Industry Classification Standard)라는 산업 분류를 사용하며, 우리나라는 표준 산업 분류를 사용합니다. GICS는 대분류와 소분류(인더스트리)로 구성되어 있습니다.
            *   Yfinance 패키지는 이러한 산업 분류 정보를 제공하여 특정 산업 분류에 속하는 회사들을 파악할 수 있게 합니다.

3.  **공식 문서를 활용한 ChatGPT 프롬프트 작성:**
    *   Yfinance 공식 문서에서 필요한 데이터(예: 섹터 및 인더스트리) 관련 설명과 예시 코드를 복사합니다.
    *   ChatGPT에게 다음과 같이 요청하여 코드를 확장합니다:
        "이 패키지에서 주는 섹터 인더스트리 자료의 예시 코드인데 이것도 포함해서, 제가 티커를 주면 주가와 산업 분류, 사업 내용 요약, 재무제표를 표로 정리하는 코드를 만들어 주세요."
    *   ChatGPT는 기존에 학습한 내용과 함께 제공된 예시 코드를 종합하여 확장된 기능을 수행하는 코드를 생성합니다.
    *   이러한 방식으로, ChatGPT가 기존에 알지 못하거나 업데이트되지 않은 정보를 외부 웹페이지나 공식 문서에서 직접 가져와 제공함으로써, 보다 정확하고 확장된 코드를 얻을 수 있습니다.

**핵심 정리**

이 시간을 통해 여러분은 코딩에 대한 전문 지식이 없더라도 다음을 할 수 있게 됩니다.

*   코랩을 활용하여 데이터를 불러오는 방법을 이해합니다.
*   야후 파이낸스와 같은 패키지의 근원이 되는 공식 사이트를 찾아가 어떤 데이터를 제공하는지 확인합니다.
*   공식 문서에서 필요한 정보(설명, 예시 코드)를 복사하여 ChatGPT에 제공하고, 원하는 기능을 포함하는 코드를 생성하도록 요청합니다.
*   이 방법은 자동 매매 봇 제작 시 증권사에서 제공하는 API 문서를 활용하여 코드를 만들 때도 동일하게 적용될 수 있습니다.

코드가 어렵다고 생각하기보다는 ChatGPT를 활용하여 필요한 코드를 쉽고 간단하게 만들어 내는 방법을 기억하시고, 앞으로 투자 분석에 적극적으로 활용하시기 바랍니다.
